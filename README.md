# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is enabled on this template. See [this documentation](https://react.dev/learn/react-compiler) for more information.

Note: This will impact Vite dev & build performances.

# DiigiiHost Portfolio Website

A modern, responsive single-page portfolio website built using **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

The website showcases the founder's journey, philosophy, brands, testimonials, and mission with smooth animations and responsive design.

---

## 🚀 Features

- Responsive UI (Desktop, Tablet & Mobile)
- Smooth scroll animations using Framer Motion
- Modern landing page design
- Tailwind CSS styling
- Auto-scrolling testimonials
- Brand showcase section
- Founder journey section
- Mission section
- Optimized image assets
- Clean component-based architecture

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- JavaScript (ES6+)

# 📦 Prerequisites

Before running this project, make sure you have installed:

- Node.js (v18 or later recommended)
- npm (comes with Node.js)

Check installed versions:

```bash
node -v
npm -v
```

---

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/RavinaMore1121/DummyDiigiihost.git
```

Go inside the project

```bash
cd DummyDiigiihost
```

Install dependencies

```bash
npm install
```

---

# ▶️ Run the Project

Start the development server

```bash
npm run dev
```

Vite will start the application.

Example:

```
Local: http://localhost:5173/
```

Open the above URL in your browser.

---

# 🏗️ Build for Production

```bash
npm run build
```

---

# 👀 Preview Production Build

```bash
npm run preview
```
